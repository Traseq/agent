服務中的任何內容均不構成：

- 投資建議或財務建議

## 2. 假設性與模擬輸出

回測、模擬、模型輸出、最佳化結果與比較分析皆為假設性結果。

它們由歷史資料、假設、參數與模型邏輯產生，不代表實際交易，也不應被呈現或解讀為實際達成的績效。

實際結果可能與假設或模擬結果有重大差異。

## 3. 回測與模擬的重要限制

假設性結果可能無法捕捉或可能低估重要真實世界因素，包括：

- 資料品質問題、缺漏資料、過期資料或倖存者偏差
- 佣金、手續費、資金費率、借貸成本、稅務與其他交易成本
- 滑價、價差變化、市場衝擊與流動性限制
- 延遲、中斷、路由差異、拒單與特定交易場所行為
- 營運故障、人為錯誤與執行流程差異
- 市場結構變化、 regime shift 與模型隨時間失穩

因此，強勁歷史結果不表示策略會在任何實盤或未來環境中表現相似。

## 4. 不保證未來結果

過去績效、模擬績效與假設性績效不代表未來結果。

獲利回測、有利模擬或強勢比較報告，不表示策略會在目前或未來市場條件下獲利、可靠、適合或安全。
