本隱私權政策說明 `Traseq`（下稱 `Traseq`、`我們`；無論目前是以個人經營者
或已登記事業名義營運）在您使用 Traseq 網站、應用程式、工作空間、API、支援管道、
AI 輔助功能與相關服務（合稱 `服務`）時，如何蒐集、使用、揭露與處理個人資料。

## 1. 適用範圍

本政策適用於下列情況下我們處理的個人資料：

- 您建立或使用 Traseq 帳號；
- 您加入、管理或被邀請加入工作空間；
- 您訂閱付費方案或購買 credits；
- 您使用 API 或 AI 輔助功能；
- 您聯絡支援、提交回饋或索取資訊；或
- 您瀏覽我們的網站或文件。

本政策不適用於第三方網站、服務或付款流程；該等情境應以第三方自己的隱私政策為準。

## 2. 我們蒐集的個人資料

我們會直接向您、您的工作空間管理者或協作者、整合供應商，以及從您使用服務的過程中蒐集個人資料。

若您未提供建立帳號、身份驗證、計費、工作空間存取或支援處理所必需的個人資料，
我們可能無法提供相應服務或其部分功能。

### A. 帳號與身份資料

可能包括：

- 姓名；
- email；
- 身份驗證供應商識別碼；
- avatar 或個人圖片；
- 登入與 session 資訊；以及
- 帳號偏好設定與 onboarding 狀態。

### B. 工作空間與協作資料

可能包括：

- 工作空間名稱與設定；
- 成員與邀請紀錄；
- 角色與權限指派；
- 組織與帳務管理資料；以及
- 管理存取與使用量所需的工作空間活動紀錄。

### C. 研究與產品使用資料

可能包括：

- 策略、版本、積木、prompt、註解與筆記；
- 回測設定、結果、分析與比較集合；
- API key 中繼資料，例如名稱、scope、建立者、狀態、最後使用時間與到期時間；
- credits 消耗、方案限制與席位配置等使用量資料；以及
- 儲存在產品、瀏覽器儲存空間或連動服務中的設定。

我們設計 API key 機制時，明文 secret 僅在建立當下顯示，不會以明文保存於資料庫。

### D. 計費與交易資料

可能包括：

- 訂閱等級、計費週期、狀態、發票與續約資訊；
- 來自付款供應商的交易識別碼；
- 稅務相關帳務紀錄；以及
- 防詐、對帳與支援所需之交易中繼資料。

若付款由第三方處理，我們通常不會有意保存完整信用卡或等效付款憑證。

### E. 溝通與支援資料

可能包括：

- 支援請求與往來內容；
- 回饋提交內容，包括分類、主旨、訊息、page URL 與除錯中繼資料；
- enterprise 聯絡表單內容；以及
- email 發送與通知紀錄。

### F. 技術、裝置與安全資料

可能包括：

- IP 位址；
- 瀏覽器、裝置與作業系統資訊；
- 請求日誌、時間戳與識別碼；
- 錯誤與效能事件；
- cookie 與 session 資訊；以及
- 用於偵測濫用、詐欺或未授權存取之資料。

### G. AI workflow 資料

若您使用 AI 輔助功能，我們可能處理：

- 您提交的 prompt 與指令；
- 生成輸出所需的工作空間上下文；
- AI 產生的策略草稿、改寫內容、分析與相關中繼資料；以及
- 維運、保護與支援該功能所需的模型使用遙測資料。

除非確有必要且您具合法權限，請不要在 prompt 中放入敏感個資。

## 3. 我們如何使用個人資料

我們使用個人資料來：

- 提供、運作、保護與維護服務；
- 驗證身份並管理帳號、工作空間、權限與 API 存取；
- 處理訂閱、購買、發票、退款與帳務事件；
- 提供回測、分析、AI 輔助輸出與其他產品功能；
- 就帳號、工作空間、計費與支援事項與您聯絡；
- 監控可靠性、除錯並改善效能與安全；
- 調查濫用、詐欺、違反條款或法律爭議；
- 履行法定義務、稽核、執法要求與留存義務；以及
- 保護 Traseq、使用者及他人的權利、安全與服務完整性。

我們通常透過電子系統與雲端服務處理個人資料，並在支援、安全、計費、法令遵循或爭議處理所需時，
由經授權人員或服務供應商進行有限度的人工作業處理。

若您位於 EEA、瑞士或英國，我們通常依下列法律依據處理個人資料：

- 契約履行；
- 我們的正當利益，例如安全、產品營運、濫用防制與客戶支援；
- 法令遵循；以及
- 在法律要求時取得您的同意。

## 4. 我們如何揭露個人資料

我們可能向下列對象揭露個人資料：

- 依據角色權限可存取資料的 workspace owner、admin 與 member；
- 協助身份驗證、託管、儲存、email 發送、計費、觀測、客戶支援與 AI 功能的服務供應商或子處理者；
- 處理訂閱、發票與購買流程的付款供應商；
- 專業顧問、會計師、稽核人員、保險人與法律顧問；
- 法律要求或為保護權利與安全而需揭露之執法機關、監理機關、法院或其他第三方；以及
- 涉及合併、收購、融資、重整或資產出售之交易對象。

以本政策生效日為準，我們不知情地將個人資料以金錢對價出售，亦不將個人資料用於第三方
跨情境行為廣告。

## 5. Cookies、Local Storage 與類似技術

我們使用 cookies 與類似技術進行身份驗證、session 管理、安全控管與產品功能運作。
我們也會使用瀏覽器儲存空間，例如 `localStorage` 與 `sessionStorage`，保存介面偏好、
待接受邀請流程、工作空間上下文、付款返回狀態與其他產品設定。

若法律要求，我們會在使用非必要 cookies 或類似技術前取得同意。您也可透過瀏覽器設定控制
部分 cookies，但停用後可能影響功能。

## 6. AI 供應商與自動化處理

當 AI 輔助功能啟用時，prompt 與相關工作空間上下文可能被傳送至我們使用的第三方模型供應商，
以生成您要求的輸出。該等供應商可能依我們的指示與其適用服務承諾處理資料。

AI 輸出具有機率性，可能不正確或不完整。最終的人工作業審查仍由您負責。

## 7. 國際資料傳輸

我們可能在您所在國家以外處理與儲存個人資料。若法律要求，我們會使用適當保障措施，
例如契約性保護或其他合法的跨境傳輸機制。

## 8. 保存期間

我們會在合理必要期間內保存個人資料，以便：

- 提供服務；
- 維持帳號歷史與工作空間延續性；
- 執行條款與處理爭議；
- 履行稅務、會計、法律與稽核義務；以及
- 防止詐欺、濫用與安全事件。

實際保存期間會依資料類型、敏感程度、您的工作空間與帳務狀態、法定要求，以及刪除或匿名化
在技術與法律上是否可行而異。

若您要求刪除，我們仍可能在法律遵循、安全、防詐、備份完整性或爭議處理所必要範圍內，
保留部分資料。

## 9. 您的權利與選擇

依您所在地法律，您可能享有以下權利：

- 知悉我們是否處理您的個人資料；
- 存取或取得個人資料副本；
- 更正不正確或不完整之個人資料；
- 請求刪除個人資料；
- 反對或限制特定處理；
- 在以同意為基礎之處理中撤回同意；
- 在適用情況下，以可攜格式取得資料；
- 在法律要求時拒絕特定用途；以及
- 向主管機關申訴或提出救濟。

### 台灣

若適用台灣《個人資料保護法》，資料當事人通常得依該法第 3 條行使權利，包括查詢或請求閱覽、
請求製給複製本、請求補充或更正、請求停止蒐集處理利用，以及請求刪除，但仍受法定例外限制。

### EEA、瑞士與英國

若適用 GDPR 或同等法律，您可能享有 GDPR 第 15 至 22 條之權利。在適用該時程時，
我們通常目標於一個月內回覆，但得依法延長。

### 加州

若適用加州法律，您可能享有知情、存取、更正與刪除特定個人資訊之權利，且不會因行使適用的
隱私權而受到違法差別待遇。

若要行使權利，請聯絡 `support@traseq.com`。我們可能需要先驗證您的身份。部分資料也可由 workspace
管理者透過產品介面存取、更正或移除。

## 10. 資訊安全

我們採用行政、技術與組織性措施來保護個人資料，包括存取控制、具 scope 的 API key、傳輸安全、
記錄與安全監控。

但沒有任何系統能保證絕對安全。您仍應自行保護憑證、妥善設定工作空間權限，並在懷疑帳號或 key
遭入侵時立即通知我們。

## 11. 兒少資料

本服務不是為 18 歲以下兒少設計，我們也不會在知情情況下蒐集 18 歲以下兒少的個人資料。
若您認為有兒少提供資料給我們，請與我們聯絡，以便我們評估並採取適當措施。

## 12. 政策更新

我們可不時更新本隱私權政策。若有重大變更，我們會公布更新版本並更新生效日。若法律要求，
我們也會提供額外通知或取得同意。
