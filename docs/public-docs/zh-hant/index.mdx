Traseq 文件把產品操作、工作空間協作與 Public Agent API 參考資料整合在同一個
Mintlify 站點。

## 開始使用 Traseq

- [在 App 中開始免費使用][docs-home-start-free-zh]

## 從哪裡開始

- 第一次使用產品：先看 [快速開始](/zh-hant/guides/quickstart)，再讀
  [核心概念](/zh-hant/guides/core-concepts)。
- 想快速理解操作流程：看
  [介面總覽](/zh-hant/guides/interface-overview) 與
  [執行第一次回測](/zh-hant/guides/tutorials/first-backtest)。
- 想接 API：從 [API 總覽](/zh-hant/api-reference/overview) 與
  [驗證方式](/zh-hant/api-reference/authentication) 開始。
- 想管理團隊空間：看
  [工作空間總覽](/zh-hant/guides/workspace/overview)。

## 目前範圍

- 目前已提供核心 onboarding、tutorial 與 workspace guides。
- `zh-Hant` 已對齊英文的四個頂層 tab。
- Public Agent API 規格預期由 backend 匯出並由 CI 驗證同步。

## Tabs

- `Guides`：產品上手、教學與工作空間文件
- `API Reference`：驗證方式、API keys 與生成式端點文件
- `Reference`：穩定的參考資料與 FAQ
- `Changelog`：產品與 API 更新
