## 目前已經搬了哪些內容？

目前站內已包含核心 onboarding、tutorial、workspace 與 API 文件，以及生成式端點文件。

## 有支援繁體中文嗎？

有。站點已提供 `zh-Hant` 導航結構，與英文維持相同的頂層 tab。

## Traseq 有提供 API 嗎？

有。Public Agent API 可讓外部工具與 AI agent 串接 workspace，並以程式化方式建立策略、執行回測與讀取結果。所有方案都可使用 Public Agent API；Free 與 Plus 受較低的 API key quota、research credits 與 workspace limits 約束。請見 [API Overview](/zh-hant/api-reference/overview)。

## API Reference 使用哪個 OpenAPI 檔案？

`openapi/traseq-public-agent.json`

這個檔案預期由 backend Swagger 匯出，而不是手工維護。
