登入 Traseq 後，你會使用以工作空間為中心的介面，包含側邊導覽、策略編輯器與分析頁面。

## 側邊導覽列

左側導覽列包含主要產品區塊：

| 項目           | 功能                     |
| -------------- | ------------------------ |
| **Home**       | 工作空間儀表板與快速操作 |
| **Strategies** | 策略建立、編輯與管理     |
| **Backtests**  | 回測歷史與狀態           |
| **Analysis**   | 比較集合與多回測分析     |
| **Settings**   | 成員、帳務與偏好設定     |

## 工作空間切換器

導覽列頂部有工作空間切換器，可在不同工作空間間快速切換。

## Home 儀表板

Home 頁面會顯示：

- 快速操作
- 正在進行中的工作
- Credits 與限制
- 最新結果

## 策略列表頁

Strategies 頁面提供：

- 名稱搜尋
- 狀態篩選
- 排序控制
- Strategies 與 Blocks 分頁

## 策略編輯器

Canvas 編輯器主要有三個區域：

| 區域              | 位置 | 功能               |
| ----------------- | ---- | ------------------ |
| **積木庫**        | 左側 | 瀏覽與加入積木     |
| **Canvas**        | 中央 | 拖放與合併策略邏輯 |
| **Entry/Exit 區** | 下方 | 定義進出場規則     |

## 回測結果頁

回測完成後，結果頁包含：

- **Summary**
- **Rules**
- **Trades**
- **Analytics**
- **Charts**

## 比較頁

Analysis 頁面可管理比較集合。進入比較模式後：

- 頂部卡片代表已選回測
- 可指定一個 baseline
- 下方分頁比較績效、風險、條件與時間區段
