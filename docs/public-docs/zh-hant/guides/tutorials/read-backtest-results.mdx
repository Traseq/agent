---
title: 解讀回測結果
description: 理解 Summary、Rules、Trades、Analytics 與 Charts 五個分頁。
---

回測完成後，結果頁會協助你檢查績效、風險與交易行為。

## Summary

Summary 分頁會顯示最重要的績效指標。

| 指標                | 說明                                       | 如何解讀                           |
| ------------------- | ------------------------------------------ | ---------------------------------- |
| **Return %**        | 回測期間的總損益                           | 正數代表獲利，越高越好             |
| **Max Drawdown**    | 帳戶價值從高點掉到之後低點的最大跌幅       | 越小越好，代表最壞情況的虧損       |
| **Total Positions** | 交易筆數（這裡的部位與交易是同一件事）     | 太少可能條件太嚴，太多可能過度交易 |
| **Win Rate**        | 獲利交易的比例                             | 要搭配獲利因子一起看，不能只看勝率 |
| **Sharpe Ratio**    | 把報酬對照「過程有多顛簸」（波動）         | 大於 1 不錯、大於 2 算強，越高越穩 |
| **Profit Factor**   | 總獲利 ÷ 總虧損                            | 大於 1 代表賺的比賠的多            |

> Note: 想看任一指標的一句話定義，請見 [術語小辭典](/zh-hant/reference/glossary)。

## Rules

Rules 分頁會顯示：

- 進場條件
- 出場條件
- 進場動作
- 出場動作

## Trades

Trades 分頁列出每一筆交易。

| 欄位            | 說明          |
| --------------- | ------------- |
| **Entry Time**  | 開倉時間      |
| **Exit Time**   | 平倉時間      |
| **Side**        | Long 或 Short |
| **Entry Price** | 進場價格      |
| **Exit Price**  | 出場價格      |
| **PnL**         | 損益金額      |
| **PnL %**       | 損益百分比    |
| **Duration**    | 持倉時間      |

## Analytics

Analytics 分頁通常包含：

- P&L Histogram（損益分佈）
- Duration Distribution（持倉時間分佈）
- MAE/MFE Scatter（每筆交易平倉前曾達到的最差／最佳未實現損益）
- Monthly Returns Heatmap（月度報酬熱圖）

## Charts

Charts 分頁會整合：

1. 價格 K 線
2. 權益曲線
3. 回撤深度
4. 交易標記

> Note: 視覺化檢查往往能看出單看總結數字不容易發現的問題。

## 快速操作

底部浮動操作列包含：

- **Run Again**
- **Compare**
