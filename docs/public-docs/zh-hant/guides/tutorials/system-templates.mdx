title: 使用系統模板快速上手
description: 從 Traseq 預先提供的模板開始，再逐步改成自己的策略。
---

如果你還不確定如何設計策略，系統模板是很好的起點。

## 瀏覽模板庫

1. 建立新策略時選擇 **From Template**
2. 或直接進入 **System Strategies**

每張模板卡會顯示：

- 策略名稱
- 簡短描述
- 核心指標或邏輯

## 載入模板

1. 點選模板卡片
2. 系統會建立一個已填入下列內容的新策略：
   - 策略名稱
   - 進場條件
   - 出場條件
   - 相關設定
3. 模板會在 canvas editor 中開啟

## 自訂與修改

載入模板後，你可以：

- 調整參數
- 增減條件
- 修改進出場動作
- 重新命名

> Note: 建議先跑一次 baseline backtest，再開始調整參數。

## 模板也是學習工具

模板能幫你理解不同指標如何組合，以及參數選擇如何影響結果。

## 建議流程

```text
選擇模板 -> 執行 baseline backtest -> 理解各指標
-> 微調參數 -> 比較版本 -> 發展自己的策略
```
