---
## 建立新工作空間

1. 點選左側導覽列頂部的 **Workspace Switcher**
2. 選擇 **Create Workspace**
3. 輸入工作空間名稱
4. 可選填描述
5. 確認建立

## 切換工作空間

1. 點選 **Workspace Switcher**
2. 選擇要切換的工作空間
3. 介面會重新載入該工作空間資料

> Note: 策略、回測與分析頁都會依目前選到的工作空間顯示內容。

## 工作空間資訊

每個工作空間可以設定：

- 名稱
- 描述
- Logo

可在 [工作空間設定](/zh-hant/guides/workspace/settings) 修改。
