工作空間是 Traseq 的研究協作單位。每個工作空間都是獨立環境，擁有自己的策略、
回測、積木、協作者與訂閱。

## 工作空間架構

| 資源         | 說明                         |
| ------------ | ---------------------------- |
| **策略**     | 工作空間內建立的所有交易策略 |
| **回測**     | 回測記錄與結果               |
| **積木**     | 自訂語義積木庫               |
| **比較集合** | 已保存的回測比較集合         |
| **協作者**   | Research seats 與 viewers    |
| **訂閱**     | 獨立的方案與預算             |

## 多工作空間

你可以用不同工作空間處理：

- 個人研究
- 共享檢視
- 教學示範

## 工作空間限制

| 方案    | 最大擁有空間數 |
| ------- | -------------- |
| Free    | 1              |
| Pro     | 1              |
| Team    | 3              |

> Note: 被邀請加入的工作空間不計入你擁有的工作空間數量。
