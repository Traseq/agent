---
title: 邀請成員
description: 邀請協作者加入 Traseq workspace，並追蹤邀請狀態。
---

Owner 與 Admin 可以邀請其他人加入工作空間。

## 發送邀請

1. 進入 **Settings** -> **Invitations**
2. 點選 **Invite Member**
3. 輸入收件者 email
4. 選擇角色：`Admin`、`Member`、`Viewer`
5. 點選 **Send Invitation**

## 邀請狀態

| 狀態         | 說明             |
| ------------ | ---------------- |
| **Pending**  | 已送出，等待接受 |
| **Accepted** | 已接受並成為成員 |
| **Expired**  | 邀請已過期       |
| **Revoked**  | 邀請已被撤銷     |

## 接受邀請

1. 打開 email 裡的邀請連結
2. 登入 Traseq
3. 確認接受
4. 系統會自動加入工作空間

## 撤銷邀請

1. 找到待處理的邀請
2. 點選 **Revoke**
