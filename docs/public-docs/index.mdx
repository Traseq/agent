Welcome to the Traseq documentation. Here you'll find everything you need to
build trading strategies, run backtests, and collaborate with your team.

## Choose your path

- **New to Traseq?** Start with the [Quickstart](/guides/quickstart), then
  read [Core Concepts](/guides/core-concepts) to understand the fundamentals.
- **Exploring the workflow?** Check out the
  [Interface Overview](/guides/interface-overview) and
  [Run Your First Backtest](/guides/tutorials/first-backtest).
- **Building an integration?** Head to the
  [API Overview](/api-reference/overview) and
  [Authentication](/api-reference/authentication).
- **Setting up a team?** See
  [Workspace Overview](/guides/workspace/overview).

- **Reference** — FAQ, glossary, and lookup material
