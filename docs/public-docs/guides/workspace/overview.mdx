A workspace is your team's home in Traseq. Each workspace is completely
independent — with its own strategies, backtests, blocks, members, and
subscription plan.

## Workspace structure

| Resource            | Description                                     |
| ------------------- | ----------------------------------------------- |
| **Strategies**      | All trading strategies created in the workspace |
| **Backtests**       | Backtest records and results                    |
| **Blocks**          | Your custom semantic block library              |
| **Comparison Sets** | Saved backtest comparison sets                  |
| **Collaborators**   | Research seats and viewers                      |
| **Subscription**    | Independent plan and budget                     |

## Why use multiple workspaces?

Separate workspaces are useful for keeping different activities organized:

- **Personal research** — your own experiments and ideas
- **Team collaboration** — shared strategies and review
- **Teaching** — demos and educational content

## Workspace limits

| Plan    | Workspaces you can own |
| ------- | ---------------------- |
| Free    | 1                      |
| Pro     | 1                      |
| Team    | 3                      |

> Note: Workspaces you've been invited to don't count against this limit.
