---
title: Create and Switch Workspaces
description: How to create a new workspace and switch between workspaces.
---

## Create a new workspace

1. Click the **Workspace Switcher** at the top of the sidebar.
2. Select **Create Workspace**.
3. Enter a name for your workspace.
4. Add a description if you like.
5. Confirm to create it.

## Switch between workspaces

1. Click the **Workspace Switcher**.
2. Pick the workspace you want to use.
3. The page refreshes to show that workspace's content.

> Note: Everything you see — strategies, backtests, analysis — belongs to the
> currently selected workspace.

## Edit workspace details

You can update your workspace's name, description, and logo in
[Workspace Settings](/guides/workspace/settings).
